1. MCQ -Multiple choice questions & MS- Multiple Select

  • Required solution components: Question Code, Solution ID, Answer, Explanation
  • Add or remove ‘Explanation’ sections as needed.
  • Use only MCQ or MS in the question template heading.
  • Change the answer option identifier as given in the question (letter/number in upper or lowercase)

2. CL - Classify

  • Required solution components: Question Code, Solution ID, Category Name, Answer, Explanation.
  • Add category 'Name', ‘Answer’, and ‘Explanation’ sections as needed.
  • Category name- The name of the category in which the given entities will be put.
  • Category Answer- The entity/s that will be put in that particular category.
  • Category Explanation- The justification for putting a particular entity in that group/category needs to be given.

3a. TCQ (Non TF) - Two Choice Question

  • Required solution components: Question Code, Solution ID, Answer, Wrong Answer, Explanation.
  • Answer (Option 1)- Put the correct answer
  • Wrong Answer (Option 2)- Put the name of the incorrect answer
  • Explanation- Justify as to why option 1 is the correct answer

3b. TCQ (TF)- True-False

  • Required solution components: Question Code, Solution ID, Answer and Explanation.
  • Answer- Simply write True or False
  • Explanation- Give justification for claiming the statement as true or false, whichever is applicable

4a. FIB - Fill in the Blank (with single blank)

  • Required solution components: Question Code, Solution ID, Answer, Explanation.
  • Answer- Put your answer that is suppose to go in the blank space.

4b. FIB- (with multiple blanks/answers and each answer with its own explanation)

  • Required solution components: Question Code, Solution ID, Answer, Explanation.
  • Add or remove ’Answer’ & ‘Explanation’ sections as needed.
  • Answer- Put your answer that is suppose to go in the blank space.
  • Explanation- Justify your answer

4c. FIB- (with multiple answers/blanks and all answers with only one explanation)

  • Required solution components: Question Code, Solution ID, Answer, Explanation.
  • Answer- Put your answer that is suppose to go in the blank space.
  • Explanation- Justify your answer
  • Add or delete ‘Answer’ as per the blank spaces in the question

5. ORD - Ordering

  • Required Solution component: Question Code, Solution ID, Answer, Explanation.
  • Add ‘Answer’ and ‘Explanation sections as needed.
  • Simply, give the correct order as answer 1, 2, 3 and so on.

6. MAT - Matching

  • Required Solution components: Question Code, Solution ID, Answer, Explanation.
  • Add ‘Answer’ and ‘Explanation’ sections as needed.
  • Answer 1 in this question type will simply be the answer that matches with the first entity in column A or 1.

7a. GR- Simple (one step)

  • Required Solution components: Question Code, Solution ID, Answer or Sample Response, graph script, explanation
  • Choose either answer or Sample Response. In case the solution includes sample response, delete the word ‘Answer’ in the question template and vice versa.
  • Answer/sample response- add the screenshot of the graph that is generated using jsx graph maker.

7b. GR- (step-by-step)

  • Required Solution components: Question Code, Solution ID, Answer or Sample Response, graph script, Step-by-step
  • Choose either answer or Sample Response. In case the solution includes sample response, delete the word ‘Answer’ in the question template and vice versa.
  • Each step includes two parts: a description of what is occurring in the step and calculations needed to solve the step
  • Add Step-by-step sections as needed (minimum 2).
  • Note: Graph script in ‘step-by-step’ section is optional. In case no additional graphs are there in ‘step-by-step’ section, delete the rows.

8a. DRW (Simple-single step)

  • Required Solution components: Question Code, Solution ID, Answer & Explanation
  • In Answer, add the screenshot of the image along with the svg file/image
  • In explanation, give justification surrounding the image drawn.

8b. DRW (Multi-step)

  • Required Solution components: Question Code, Solution ID, Answer & steps
  • In Answer, add the screenshot of the image along with the svg file/image
  • Add Step-by-step sections as needed (minimum 2).
  • In Answer, add the screenshot of the image along with the svg file/image
  • In explanation, give justification surrounding the image drawn.

9a. LDGR (simple)

  • Required Solution component: Question Code, Solution ID, Answer, Explanation
  • Answer- Add screenshot of the ledger
  • Answer- add link of the Google sheet where ledger is created.

9b. LDGR- Step-by-Step (only 1 spreadsheet link)

  • Required Solution component: Question Code, Solution ID, Answer, Explanation
  • Answer- Add screenshot of the ledger
  • Answer- add link of the Google sheet where ledger is createdAdd Step-by-step sections as needed (minimum 2).

9c. LDGR- Step-by-Step (more than 1 spreadsheet link)

  • In such a case where more than 1 spreadsheet links are to be given, use the format for SA question
  • Add ‘step-by-step’ section according to the question’s requirement.


10a. SA - Short Answer (one step)

  • Required Solution component: Question Code, Solution ID, Answer, Explanation
  • Choose either answer or Sample Response. In case the solution includes sample response, delete the word ‘Answer’ in the question template.

10b. SA - Short Answer(Step by Step)

  • Required Solution component: Question Code, Solution ID, Answer, Explanation
  • Choose either answer or Sample Response. In case the solution includes sample response, delete the word ‘Answer’ in the question template.

11. ES - Essay

  • Required Solution components: Question Code, Solution ID, Answer or Sample Response, Explanation
  • In ideal cases, for an open-ended answer that comprises only text, ES template has to be used.
  • In special cases, where there is a graph or ledger along with some text as a part of answer, the question template should be switched to ES type instead of LDGR and/or GR.

12a. CALC-Calculation (One step Solution)

  • Choose either answer or Sample Response. In case the solution includes sample response, delete the word ‘Answer’ in the question template and vice versa.
  • Do not use this template simply to merge steps which might actually require step-by-step question template.

12b. CALC (step-by-step)

  • Required Solution components: Question Code, Solution ID, Answer, Step-by-step
  • Each step includes two parts: a description of what is occurring in the step and calculations needed to solve the step.
  • Add ‘Step-by-step’ sections as needed (minimum 2).
  • Choose either answer or Sample Response. In case the solution includes sample response, delete the word ‘Answer’ in the question template.

13. Modified SA (For Proof based questions)

  • Required Solution components: Question Code, Solution ID, Entire Proof Answer, Step-by-step
  • Each step includes two parts: a description of what is occurring in the step and calculations needed to solve the step.
