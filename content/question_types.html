Question Types

This page provides instructions for authoring solutions for different question types.

MCQ | MS | TCQ (TF)| FIB | SA | ES | CALC | ORD | GR | DRW | LDGR | MAT | CL

Multiple Choice (MCQ)

A multiple choice question is a problem that has a prompt and provides several possible discrete answer options where only 1of the provided answer options is the correct answer.
Note: If more than one answer option could be selected as a response, the MS question type solution should be used. If there are only two answer options, the TCQ question type solution should be used.

Example:

Which of the following is a type of plant?
A) fern
B) canine
C) mushroom
D) shark

There is one and only one correct response to this question: A) fern

Directions for authoring MCQ solutions:

There are always three types of <section> blocks present in an MCQ solution:

  • answer
  • explanation
  • wrong_explanation

For the first <answer> section, the solution is provided and the letter of the correct answer option is provided in the format that matches the textbook (uppercase vs. lowercase).

For the second <explanation> section, the explanation for the correct answer option is provided to the student. Provide an explanation for why this is the correct answer. The explanation should be relevant to the overall question and help a student better understand the concept being tested. Do not include the text of the stem or the answer option (as it appears in the textbook) in your solution.

For the third and all subsequent <wrong_explanation> sections, the explanations for the incorrect answer options are provided to the student. There will be a wrong_explanation section block provided for each incorrect answer option. The explanation should be relevant to the overall question and help a student better understand the concept being tested. Do not include the text of the stem or the answer option (as it appears in the textbook) in your solution.

Multiple Select (MS)

A multiple select question is a problem that has a prompt and clearly provides several possible answer options where more than 1 of the answer options could be a correct answer. Note: some MS questions can have a single answer option keyed, but the question prompt allows for more than one answer option to be selected (e.g., "Select all the apply.").

Example:

Which of the following are plants? Select ALL that apply.
A) fern
B) canine
C) mushroom
D) tree

There are two correct answer options that should both be keyed as part of the correct response: Option (A) and Option (D).

Directions for authoring MS solutions:

There are always three types of <section> blocks present in an MS solution:

  • answer
  • explanation
  • wrong_explanation

For an MS question, there can be multiple answer <section> blocks depending on how many answer options are correct. Each answer <section> block includes the letter of a correct answer option (provided in the format that matches the textbook, uppercase vs. lowercase).

For the next <section> blocks, an explanation for each correct answer option is provided to the student.

For the final <section> blocks, the explanations for the incorrect answer options are provided to the student. There will be a <wrong_explanation> section block provided for each incorrect answer option.

Two Choice (TCQ)

A two choice question is a problem that has a prompt and provides two possible discrete answer options where only 1 of the provided answer options is the correct answer. This question type should be used for TF.

Non-TF Example:

For the following cost, determine whether it is a fixed or variable cost:
Purchase of a new truck

There is one and only one correct response for this question:
fixed

Directions for authoring TCQ solutions:

There are three types of <section> blocks present in a non-TF TCQ solution:

  • answer
  • wrong_answer
  • explanation

For the first <section> block, the correct answer is provided, this should generally be no longer than a couple words.

For the second <section> block, the incorrect answer option is provided to the student, similar to the previous section.

For the third <section> block, the explanation for the correct & incorrect answer options is provided to the student.

True or False (TF) Example:

A True or False TCQ question is a problem for which the correct answer can only be true or false.

Is this statement true or false? A fern is a type of plant.

The correct response is "true."

Directions for authoring TF solutions:

There are always two types of <section> blocks present in a TF solution:

  • answer
  • explanation

For a TF question, there is a single answer <section>. The answer can only be "true" or "false".

For the second <section>, an explanation for the answer (true or false) is provided to the student.

Fill-in-the-Blank (FIB)

A Fill-in-the-Blank problem is a statement that includes a blank space (or spaces) for the answer. The statement only makes sense if the correct answer(s) is filled in the provided blank(s). Answer choices may be selected from a given set, or not given at all.

Example:

The capital of California is _____________ and the capital of Colorado is _____________.

The correct response is Sacramento and Denver.

Directions for authoring FIB solutions:

There are always two types of <section> blocks present in a FIB solution:

  • answer
  • explanation

For a FIB question, there can be multiple answer <section> blocks depending on how many blanks are present in the question stem. An answer <section> is provided for each blank.

For the next <section> blocks, an explanation for each blank is provided to the student.

Short Answer (SA)

A Short Answer question is a problem for which an answer can be provided by writing a short response that may or may not require full sentences. SA questions may require answers in the form of definitions, names/lists, explanations, comparisons between ideas that expect a short response, or a step-by-step process. Examples of answers could be the name of a person or the date of a historical event, yes or no, or a single paragraph that contains an explanation for a specific topic. Responses should be fewer than 100 words.

Example:

What are some of the major differences between the structure of plant and animal cells?

Plant and animal cells differ in some of the following ways:

  • Chloroplasts are present in plants and some algae, but are not found in animal cells. Photosynthesis takes place in chloroplasts.
  • Cell walls are present in plants, but not in animal cells.
  • Vacuoles are membrane-bound sacs that function in storage and transport. A large central vacuole is found in plant cells, but not in animal cells. (Animal cells contain many smaller vacuoles.)
  • Centrioles are found in centrosomes and are present in animal cells, but are absent from plant cells.
  • Lysosomes are present in animal cells, but are rarely seen in plant cells. Lysosomes help digest cellular macromolecules.
  • Cilia are usually observed in animal cells, but are absent from plant cells. Cilia are microtubules that help the cell move.

Directions for authoring SA solutions:

When there is a single correct answer possible, the SA solution should include two types of <section> blocks:

  • answer
  • explanation

Answers should be direct responses to the question prompt, while explanations provide supporting information. This structure helps to reduce redundancy between the two sections. In most non-MCQ or non-MS question types, students will always see the answer and explanation paired together.

When the SA question is considered open-ended and more than one correct response is possible, the SA solution will normally include two types of <section> blocks:

  • sample response (required)
  • explanation (optional)

A sample response is often used when a question asks for an example or opinion, leaving the question highly open-ended. This format is used when the answer and explanation would be identical.

The answer, sample_response, or explanation sections can be formatted with bulleted lists or html tables to make the content more concise and easier to parse.

Essay (ES)

An essay question is a problem for which an answer can be provided by writing a more in-depth response that requires full sentences. The question may provide a prompt with a situation or topic, and should include instructions for what the essay should include (e.g, specifying a minimum word count or number of pages, requiring a summary/introduction, or specifying the format of the essay [double-spaced, APA, etc.]). The answer would typically include full sentences and consist of one or more paragraphs. Responses should be more than 100 words.

Example:

How has the discovery of DNA been important to society?

Directions for authoring ES solutions:

The ES solution will include two types of <section> blocks:

  • sample response
  • explanation

The explanation section can be formatted with a bulleted list to make the content (guidelines on how to construct the essay) more concise and easier to parse.

Calculation (CALC)

A question or problem that requires computation to solve. It may instruct the student to solve or prove it. CALC problems often include equations in the questions and/or words such as Solve for x, Calculate x, Find x, Prove that xâ€¦, How manyâ€¦ etc. The answers could be a number, a rate, or an equation that proves a math equation is true. CALC questions will usually require step-by-step solutions.

Example:

Find the value of each permutation: P(6,2)

Directions for authoring CALC solutions:

The CALC solution will normally include two types of <section> blocks:

  • answer
  • step_by_step

When a CALC question is more open-ended and more than one correct response is possible, the CALC solution will include two types of <section> blocks:

  • sample response
  • step_by_step

The first section, the answer or sample_response section, will include the numeric answer for the question.

The second section, the step_by_step section, provides clear step-by-step instructions directing students how to solve the problem to arrive at the answer. While most CALC questions will include a step-by-step solution, it's acceptable in certain cases to include an explanation section instead of a step_by_step section. This is reserved for cases where a solution does not warrant a step-by-step template (e.g., does not require a true calculation or requires a single step and doesn't warrant a step-by-step template).

Ordering (ORD)

A question for which the answer requires the ranking or sequencing of several items provided by the prompt. These questions commonly include the words Rank or Order in the question and ask for a ranking of the items based on a specific criteria.

Example:

Place the following measurements in order from shortest to longest: 1 m, 10 cm, 1,000 in, 2 ft

The correct order is: 3,1,4,2

Directions for authoring ORD solutions:

There are always two types of <section> blocks present in an ORD solution:

  • answer
  • explanation

For an ORD question, there will be multiple answer <section> blocks.

For the next <section> blocks, an explanation for each option is provided to the student.

Graphing (GR)

A question for which the answer must be a graph. Examples includes business questions that ask for a line or curve to be drawn on a provided graph or a graph of a function. Questions will sometimes include the word Graph or Label. Note that circle graphs (pie charts) or bar graphs should be treated as DRW questions.

Example:

Graph $y=2x+1$

Directions for authoring GR solutions:

The GR solution should include two types of <section> blocks:

  • answer
  • step_by_step

When a GR question is more open-ended and more than one correct response is possible, the GR solution will include two types of <section> blocks:

  • sample_response
  • step_by_step

The first section, the answer or sample_response section, will include the script for the JSXGraph.

The second section, the step_by_step section, provides clear step-by-step instructions directing students how to develop, label, and format their graphs to fully satisfy the requirements of the question. While most GR questions will incude a step-by-step solution, it's acceptable in certain cases to include an explanation section instead of a step_by_step section. This is reserved for cases where a solution does not warrant a step-by-step template.

Drawing (DRW)

A question for which the answer must be drawn by the student. Examples include questions that ask for a sketch or diagram. Questions will sometimes include the word Draw or Label. Note: graphs that cannot be developed using the JSXGraph tool will be considered DRW questions and an image should be developed for the required graph.

Example:

Draw a Punnett Square to show the genotypes that would result from a cross of two parents heterozygous for an allele (genotype Tt).

The correct response is:



  • A correct Punnett Square shows that the resulting genotypes (allele combinations) of the offspring are TT, Tt, and tt in the ratio 1:2:1.
  • To set up the Punnett Square, the parents' alleles are separated and placed across the top of the Punnett Square (for one parent) and repeated to the left of the Punnett Square (for the other parent).
  • Each cell in the Punnett Square is a possible allele combination for an offspring.

Directions for authoring DRW solutions:

When there is a single correct answer possible, the DRW solution should include two types of <section> blocks:

  • answer
  • explanation

When the DRW question is open-ended and more than one correct response is possible, the DRW solution will include two types of <section> blocks:

  • sample response
  • explanation

The first section, the answer or sample_response section will include a single image (svg) file.

The second section, the explanation section, can be formatted as a bulleted list or step-by-step template to make the content more concise and easier to parse.

Ledger (LDGR)

A question for which the answer is a ledger, journal, or accounting statement.

Example:

ABC Company creates a journal entry for June 1, 2018 detailing the companyâ€™s issuing of common stock (credit) in exchange for $100,000 in cash (debit). The cash account number is 1000, and the stock account number is 3000. Create a journal entry for the ABC Company.

Directions for authoring LDGR solutions:

The LDGR solution should include two types of <section> blocks:

  • answer
  • explanation

The first section, the answer section, will include the image (svg) file of the accounting statement. It will be created using the ledger templates provided in Google sheets and a screenshot of the ledger (without gridlines) will be uploaded as the image.

The second section, the explanation section, is commonly formatted as a step-by-step that details the steps taken to develop the ledger or statement (but is not required to be a step-by-step).

Matching (MAT)

A question where there is a set of givens and a set of choices and the question instructs you to match the givens with the choices in a 1:1 way by labelling, writing them in a space, etc.

Example:

Match each organism with the best description.

____ cat
____ trout
____ snake
____ frog
a. amphibian
b. mammal
c. reptile
d. fish

The correct answer is:

b
d
c
a

Directions for authoring MAT solutions:

There are always two types of <section> blocks present in a MAT solution:

  • answer
  • explanation

For a MAT question, there will be multiple answer <section> blocks, the total depending on how many matches (often with corresponding blanks) are being made in the question stem.

For the next <section> blocks, an explanation for each match is provided to the student.

Classifying (CL)

A question where a set of options are classified into a given group (usually two or more).

Example:

For the following organelles, classify them as organelles that are found in
I. animal cells, II. plant cells, or III. both animal and plant cells.

a. cell wall
b. nucleus
c. central vacuole
d. ribosome
e. centriole

The correct answer is:

I. Animal cells: e. centriole
II. Plant cells: a. cell wall, c. central vacuole
III. Both animal and plant cells: b. nucleus, d. ribosome

Directions for authoring CL solutions:

In the example provided here, "I. Animal cells", II. "Plant cells", and III. "Both animal and plant cells" are the group names. When determining the group names, follow these guidelines:

  1. The groups are the categories the options will be sorted into.
  2. If there is a designator used for groups in the book (e.g., I, II, III, a, b, c, or 1, 2, 3), include them with the group names.
  3. If the group name is five words or less, include the full group name from the book. Be aware that certain group names will need to be modified to avoid infringement (e.g., proper nouns). These can be abbreviated.
  4. If the group name is over 5 words, abbreviate or truncate it.

In the example provided here, "a. cell wall", "b. nucleus", etc. are the answer options. When determining what text to include for the answer options, follow these guidelines:

  1. The answer options are sorted into the groups provided in the textbook. Depending on the question, a group may have one answer option sorted into it, more than one answwer option in it, or no answer options sorted into it.
  2. If there is a designator used for the answer options in the book (e.g., a, b, c, or 1, 2, 3), include them with the anwer option.
  3. If the answer option is five words or less, include the full answer option text from the book. Be aware that certain answer options will need to be modified to avoid infringement (e.g., proper nouns). These can be abbreviated.
  4. In a question where all or some of the the answer options' text is over 5 words (e.g., full sentences), you may include just the letter designator for the answer options (and no text).

For the next <section> blocks, an explanation for each group (that discusses all the options classified into that group) is provided to the student.

Some complex CL questions require the student to classify the answer options into groups and then to perform an additional task in their response (e.g., to classify the answer options again or to provide additional information for each answer option). In these cases, the Short Answer question type template should be used instead of the Classifying template.
