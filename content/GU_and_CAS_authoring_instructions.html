General Instructions

Every graphing utility (GU) and computer algebra system (CAS) is slightly different, so it is important to focus on the core functionality that a user would utilize with any standard GU or CAS to arrive at an answer. Although each utility is slightly different, almost all of them have the same base functionality. As a result, we can author solutions that do not rely on a specific utility (such as a TI-84 calculator or Maple).

Since the goal is to write a solution that is GU/CAS agnostic, an author can utilize any major tool to understand what functionality and/or options are needed to write a quality content-accurate solution.It is important to note that an author should not indicate the GU/CAS that was utilized to test the process, nor should they reference any tool-specific language or menu/navigation features.

All GU/CAS-oriented solutions should include:

  1. which option(s)/functionality a student will utilize (GU agnostic)
  2. the order in which a user should utilize the options/functionality
  3. JSXGraph(s), whenever applicable, to show the desired output
  4. an explanation of what the results should be and how a user should interpret them to arrive at the answer
  5. the answer

Examples of an author using a GU to assess needed options/functionality

The given examples show how an author can utilize a GU to understand generic options and functions, as well as desired output to arrive at the answer.

Points of Intersection

  • Enter both equations into a graphing utility and use the INTERSECT function to find one of the $x$- and $y$-values. For some online utilities, simply highlight or hover over the point of intersection and the coordinates will be displayed.
  • Use the function again to find any other value(s).
  • Most graphing utilities are accurate to at least five decimal places. All of them are accurate to three decimal places.


Solving Equations

There are several options for solving an equation with a graphing utility:

  • Use the SOLVER function.
  • Manipulate the equation so that zero is on one side of the equal sign. Then, find the zeros of the expression that is on the other side.
  • Find the INTERSECTION of the graphs of:
    • $y= \text{(the left side)}$
    • $y= \text{(the right side)}$


Performing Regressions

  • Enter the data into a graphing utility. (Many utilities use the command EDIT to enter data.)
  • Calculate the Quadratic Regression for the data.
  • Most graphing utilities assume that the output required is the general form of a Quadratic Equation, $y=ax^{2}+bx+c$, and will return the values for $a$, $b$, and $c$.
