Finance Calculations


Finance questions can be answered using a formula, a calculator (in a simple table format, shown below), Google sheet functions and/or Google sheet table. When a question can be answered using two or more options (formula and calculator and/or Google sheet’s function, etc.), the solution should contain all the options and should be label accordingly (e.g., Option 1, Option 2, and Option 3).

In non-LDGR questions (CALC, SA, etc.), a KaTeX formula, a simple table, or a Google sheet table can be used to generate calculation steps. Refer to the Formulas page for guidelines on when to use a formula, a simple table, or a Google sheet table. A link to the Google sheet should be added to the solution if applicable.

For complex questions that require the use of a spreadsheet (Google sheets) to arrive at the answer and the final answer is a spreadsheet, the question should be treated as a LDGR question and follow the LDGR authoring guidelines. A Google sheet link should be added to the solution if applicable.


How to include a Google sheet’s function in the solution

  1. Select the correct Google sheet function for the solution (see the Google sheet function table below).
  2. Enter the appropriate values in the Google sheet function.

How to display calculator in the solution:

  1. Create the calculator table (simple table) in your solution.
  2. Enter appropriate values for each cell of the calculator.
  3. Add extra rows/columns/other key functions if needed.
