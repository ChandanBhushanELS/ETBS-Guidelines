Code

Include code in your solutions by surrounding the code block with <pre> and <code> tags. You must specify the language of your code by adding a class to the <code> tag, such as <code class="javascript">. See the list of language classes here. Code blocks should be used for any content that would need to be executed (including excel formulas) or for sample output from executed code. Code blocks can also be used when a fixed-width font is needed, with a language of "plaintext".

Example: 
<pre><code class="javascript">function add(x, y) {
    return x + y;
}
</code></pre>
Result:
function add(x, y) {
    return x + y;
}

The above code will display:

Code style is very important and should set a positive example for students. Style should follow standards set in the source, or the most common standard for the language if the source does not specify or include examples.

Gotchas

  • Make sure not to include a newline between the <code> tag and the first line of your code. This will be displayed as a blank line in your code, since everything within the <code> tags is displayed as is.
  • Make sure you don't have too many indents within your <code> tags. Some editors will attempt to insert indents automatically.
  • If you need to include code for a language not specified in the table below or full documentation, please let your manager know so we can add the language.
