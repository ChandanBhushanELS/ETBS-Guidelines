Tables

Tables that are needed as part of solutions will be built in HTML.

The following is a very simple table format that can be copied or adjusted as needed. It is a sample of an HTML table included as part of an answer section and the HTML code for the table is provided.

Entry First Last Age
1 Mark Taylor 30
2 Jacob Kim 45
3 Sarah Lee 25
